Workshop Detail

ECMAScript 6 - The Lastest JavaScript Fundamentals

ADD TO CALENDAR...

ES2015 is here! All features have been finalized. Come learn how to start using it today with new and existing projects to leverage all the new additions that ES2015 brings to the table.

REGISTER

We'll cover transpilers, build tools, caveats, and patterns that have already shown themselves with ES2015. Of course, we we also cover in detail some of the bigger features like modules, classes, arrow functions etc.

From then, we'll go over some of the new proposals for ES2016+ and how to use them. You'll leave knowing how to use ES2015 today not matter what environment you are writing your JS app (browser, node, hardware).
