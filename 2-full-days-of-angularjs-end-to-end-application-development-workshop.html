Workshop Detail

2 Full Days of AngularJS End-to-End Application Development Workshop

2 Full Days 3/25-26 9to5
ADD TO CALENDAR...

Interested in diving into AngularJS and learning how it can be used to build robust Single Page Applications (SPAs)? The AngularJS End-to-End Application Development workshop provides a hands-on look at working with the AngularJS framework.

REGISTER

The course starts with an introduction to building Single Page Applications (SPA) and talks about the features AngularJS provides. From there you’ll learn about different aspects of the framework such as views and directives, controllers and routes, as well as factories and services.

Along the way, different debugging techniques and tools are discussed, how different AngularJS components can be customized and shared with other components, and different JavaScript patterns that can be applied to applications to make them more maintainable. By the end of the workshop you’ll have walked through the process of building a Single Page Application (SPA) from end-to-end using AngularJS and be ready to apply that knowledge to applications you need to build.

Although the focus is on AngularJS, the application discussed also provides two alternative back ends including Node.js, Express and MongoDB or ASP.NET Web API with Entity Framework and SQL Server.

This is a BYOL (Bring Your Own Laptop) workshop To run the labs in the course and get the most out of it you’ll need the following software installed before the workshop begins.

  • Node.js, MongoDB, and your favorite code editor if you want to run the Node backend RESTful service
  • Visual Studio 2013 (the free Express for Web version is fine) installed with LocalDb (installed by default) if you want to run the Web API backend RESTful service
    • We won’t have time to fix installation issues during the workshop so please come prepared with all of the necessary software installed and working properly.
