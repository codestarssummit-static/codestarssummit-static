Workshop Detail

Hands on with Microsoft Azure

1 Day Workshop on Friday, September 30, 2016 (9am to 4pm)
ADD TO CALENDAR...

Think Microsoft Azure is all about Windows and .NET? Think again. In this workshop,...

  • Free Microsoft Azure Workshop (Limited Attendance)
  • Register

we'll show how you can use Azure services to build and manage scalable, reliable cloud applications using a wide variety of languages, frameworks, and tools. In particular, we'll cover how and when to use the following services:

  • Azure App Service for building websites and mobile backends
  • Azure Functions for processing events with serverless code
  • Azure Container Service for running containerized applications using DC/OS or Docker Swarm
  • Azure Service Fabric for building microservice-based applications

You will get your hands dirty with each of these services, from development to continuous integration and deployment, to production operations, monitoring, and diagnostics. What will not get dirty is your laptop, as we will provide preconfigured development VMs with all necessary SDKs and tools already installed.
